# vue-electron

## 下载依赖
```
npm install or cnpm i
```

### 编译运行 vue 项目
```
npm run serve 
```

### 编译运行桌面应用
```
npm run mock
```

### 编译 vue 项目
```
npm run build
```

## 打包项目
- 打包好的项目在 electron/app 内
```
npm run package
```
